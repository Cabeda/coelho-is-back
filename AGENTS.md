# Repository Preferences for AI Agents

## Package Manager

- **Use pnpm** for all package management operations
- Do not use npm or yarn

## Project Type

- Next.js application with App Router
- TypeScript
- Tailwind CSS for styling

## Code Style

- Retro arcade/80's game aesthetic
- Fun and vibrant colors
- Performance-conscious (limit animations, optimize renders)

## Deployment

- Target platform: Fly.io
- Database: SQLite with Prisma
- Docker with multi-stage builds
- Use standalone output for Next.js

## Development Preferences

- Clear, maintainable code
- Server Actions for database operations
- CSS transforms for animations (better performance than JavaScript)
